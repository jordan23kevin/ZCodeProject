# 胚衣参数表

> 用途：以后新增胚衣时，你只需要填这张表。告诉我后，我会读取这张表，把参数同步到 `white_t_mockup/presets.json`，再用对应胚衣出图。

## 填写说明

1. **胚衣文件名**：填实际文件名，如 `3B.png`。
2. **胚衣路径**：填完整路径，如 `D:\Semems\1胚衣\白\3B.png`。
3. **适用贴图后缀**：填 `B`、`W`、`BW`，可组合，如 `B/BW`。
4. **方法**：新胚衣一般填 `transform`；只有旧版 PSD 带手部遮罩模板填 `legacy`。
5. **缩放/旧版高度**：`transform` 填百分比数字，如 `32` 表示 32%；`legacy` 填目标高度像素，如 `677px`。
6. **旋转方向**：填 `顺时针`、`逆时针` 或 `无`。我会按内部约定转换：顺时针为正，逆时针为负。
7. **旋转角度(°)**：只填数字，如 `3`；不旋转填 `0`。
8. **最高像素点 y**：贴图有效像素（非透明区域）最高点要落到的 y 坐标。
9. **中心点 x**：贴图有效像素水平中心要落到的 x 坐标。
10. **混合模式**：目前统一填 `Multiply`。
11. **输出命名规则**：可选；不填我会按编号和后缀自动生成，如 `<编号>_B_白T.jpg`。
12. **备注**：可选。
