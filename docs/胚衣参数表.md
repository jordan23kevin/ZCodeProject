# 胚衣参数表

> 用途：以后新增胚衣时，你用 Excel 填写 `docs/胚衣参数表_模板.csv`，保存/导出为 **CSV UTF-8**（Excel 可能用分号分隔，我会兼容）后发我；我会读取它，转换成这张 `.md` 记录，并同步到 `white_t_mockup/presets.json`。

## 必填表头

| 表头 | 怎么填 |
|---|---|
| 胚衣文件名 | 实际文件名，如 `3B.png` |
| 胚衣完整路径 | 完整路径，如 `D:\Semems\1胚衣\白\3B.png` |
| 适用贴图后缀（B/W/BW） | 填 `B`、`W`、`BW`，可组合，如 `B/BW` |
| 方法（transform/legacy） | 新胚衣一般填 `transform`；旧版 PSD 带手部遮罩模板填 `legacy` |
| 缩放百分比 | 必填；如 `32` 表示 32% |
| 旋转方向（顺时针/逆时针/无） | 必填；只能填 `顺时针`、`逆时针`、`无` |
| 旋转角度（数字） | 必填；只填数字，如 `3`；不旋转填 `0` |
| 最高像素点y | 贴图有效像素（非透明区域）最高点要落到的 y 坐标 |
| 中心点x | 贴图有效像素水平中心要落到的 x 坐标 |
| 混合模式（Multiply） | 目前统一填 `Multiply` |
| 输出命名规则（可空） | 可选；不填我会按编号和后缀自动生成 |
| 备注（可空） | 可选 |
